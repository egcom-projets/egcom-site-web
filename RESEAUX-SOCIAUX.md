# 📱 Configuration des Réseaux Sociaux EGCOM-PROJETS

## 🔗 Comment mettre à jour vos liens de réseaux sociaux

### **Instructions :**

Une fois que vous aurez créé vos pages sur les réseaux sociaux, remplacez les liens dans le fichier `index.html` (et autres pages) :

---

## 📋 **Liens actuels à remplacer :**

### **1. Facebook**
- **Lien actuel** : `https://www.facebook.com/VOTRE-PAGE-FACEBOOK`
- **À remplacer par** : L'URL de votre page Facebook EGCOM-PROJETS
- **Exemple** : `https://www.facebook.com/egcomprojets`

### **2. LinkedIn**
- **Lien actuel** : `https://www.linkedin.com/company/VOTRE-PAGE-LINKEDIN`
- **À remplacer par** : L'URL de votre page entreprise LinkedIn
- **Exemple** : `https://www.linkedin.com/company/egcom-projets`

### **3. Instagram**
- **Lien actuel** : `https://www.instagram.com/VOTRE-COMPTE-INSTAGRAM`
- **À remplacer par** : L'URL de votre compte Instagram
- **Exemple** : `https://www.instagram.com/egcomprojets`

### **4. YouTube**
- **Lien actuel** : `https://www.youtube.com/@VOTRE-CHAINE-YOUTUBE`
- **À remplacer par** : L'URL de votre chaîne YouTube
- **Exemple** : `https://www.youtube.com/@egcomprojets`

### **5. Twitter (X)**
- **Lien actuel** : `https://twitter.com/VOTRE-COMPTE-TWITTER`
- **À remplacer par** : L'URL de votre compte Twitter/X
- **Exemple** : `https://twitter.com/egcomprojets`

---

## 🛠️ **Comment mettre à jour les liens :**

### **Méthode 1 : Rechercher et remplacer dans VS Code**

1. Ouvrez VS Code
2. Appuyez sur `Ctrl + H` (Rechercher et remplacer)
3. Dans "Rechercher" : `VOTRE-PAGE-FACEBOOK`
4. Dans "Remplacer par" : `egcomprojets` (ou votre vrai nom de page)
5. Cliquez sur "Remplacer tout"
6. Répétez pour chaque réseau social

### **Méthode 2 : Édition manuelle**

1. Ouvrez `index.html`
2. Cherchez la section "Suivez-nous" (ligne ~310)
3. Remplacez chaque `VOTRE-XXX` par votre vrai nom de compte
4. Sauvegardez le fichier

---

## 📝 **Noms de comptes recommandés :**

Pour une cohérence de marque, utilisez le même nom sur tous les réseaux :

- **Nom suggéré** : `egcomprojets` ou `egcom.projets`
- **Alternative** : `egcom_projets_guinee`

---

## ✅ **Après avoir créé vos comptes :**

1. Créez vos pages/comptes sur chaque réseau social
2. Notez les URLs exactes
3. Remplacez les liens dans `index.html`
4. Testez chaque lien pour vérifier qu'il fonctionne
5. Déployez avec : `git add . && git commit -m "Mise à jour liens réseaux sociaux" && git push`

---

## 🎯 **Avantages de cette configuration :**

✅ **target="_blank"** : Ouvre dans un nouvel onglet  
✅ **rel="noopener noreferrer"** : Sécurité et performance  
✅ **Prêt à l'emploi** : Il suffit de remplacer les URLs
